Recent Updates

UNECE WP.29 Amendment Updates

October 28, 2024

New

The latest amendments to UN Regulations R155 and R156 introduce enhanced requirements for cybersecurity risk assessment and software update validation processes. Key changes include mandatory threat modeling documentation and expanded incident reporting requirements.

Impact: High Effective: January 2025
Read More →

ISO/SAE 21434 Revision Draft

October 15, 2024

Updated

The draft revision of ISO/SAE 21434 includes new guidance on AI/ML security considerations in automotive systems, expanded supply chain security requirements, and updated incident response procedures for connected vehicles.

Impact: Medium Comment Period: Until Dec 2024

AUTOSAR Adaptive Platform R24-03

September 30, 2024

Released

The latest AUTOSAR Adaptive Platform release introduces enhanced security features including improved cryptographic services, secure communication protocols, and advanced intrusion detection capabilities for software-defined vehicles.
